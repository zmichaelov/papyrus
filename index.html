Papyrus

Papyrus is a text editor with word prediction built with wxPython. It was inspired by the (now cancelled) Google Labs project, Scribe.

Features

  • Tabbed editing
  • Suggestions pulled from Google's autocomplete functionality

Feature requests and bugs

Papyrus is at a very early stage in its development so feature requests and bug reports are encouraged. Feel free to file an issue on the Github issues page.
